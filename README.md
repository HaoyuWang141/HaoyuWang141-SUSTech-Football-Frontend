# SUSTech-Football-Frontend

南科大足球赛事小程序 前端仓库



## 功能

### 用户基础功能

1. 大厅：赛事、比赛、球队大厅及查询
2. 快捷功能：我是球员、我是裁判
3. 关注的比赛：跳转、关注、取消关注
4. 关注的球队：跳转、关注、取消关注
5. 关注的赛事：跳转、关注、取消关注


### 球员

1. 注册：跳转、填写信息（头像、其他）、点击注册
2. 编辑信息
3. 跳转教练页面
4. 跳转裁判页面
5. 申请加入球队
6. 我的球队
7. 我的赛程
8. 我的赛事
9. 通知、回复
